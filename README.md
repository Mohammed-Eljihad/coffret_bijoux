## 📖 Design Philosophy: "Auric Silence"

Born from the intersection of haute couture and negative space, this project explores the paradox of luxury as restraint.
- **Space & Form:** Form breathes in vast fields of charcoal (#0d0d0d), revealing objects rather than placing them.
- **Color Palette:** A monastic discipline of Charcoal (#0d0d0d), Ivory Cream (#FAF7F2), and a signature Amber-Gold (#C9A96E) accent.
- **Typography:** Dramatic oscillations between monumental display letterforms and microscopic, textural details.

## 🚀 Technical Excellence

This project is built with a modern, high-performance stack ensuring seamless user experience and premium feel.

- **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Typography:** [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) (Display) & [Jost](https://fonts.google.com/specimen/Jost) (Body)

## ✨ Key Features

- **Immersive Hero Section:** Large-scale typography and premium product imagery.
- **Dynamic Benefits & Materials:** Meticulously detailed sections showcasing quality.
- **Interactive Product Variants:** Real-time color switching with smooth transitions.
- **Testimonial Showcase:** Social proof with a premium aesthetic.
- **Interactive FAQ:** Clean, accordion-style information delivery.
- **Fully Responsive:** Meticulously crafted layouts for Desktop, Tablet, and Mobile.
- **Performance Optimized:** Static Site Generation (SSG) for instant load times.

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/coffret-bijoux.git
   cd coffret-bijoux
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🌐 Vercel Deployment

This project is optimized for deployment on [Vercel](https://vercel.com/).

1. Push your code to GitHub.
2. Import the project in Vercel.
3. Vercel will automatically detect the Next.js framework and configure the build settings:
   - **Build Command:** `npm run build`
   - **Output Directory:** `.next`
   - **Install Command:** `npm install`
